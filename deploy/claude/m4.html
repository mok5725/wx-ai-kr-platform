직무 응용 & 미니 프로젝트

이제 배운 걸 진짜 내 업무에 적용합니다. 여러 파일을 한 번에 정리·요약·변환하고, 반복 업무를 작은 도구로 만들어 봅니다. (모두 더미데이터)

🎯 이 단계를 마치면
내 반복 업무 1건의 처리 흐름을 설계할 수 있습니다.

1. 먼저 "반복되는 일"을 찾으세요

클로드 코드가 가장 큰 힘을 내는 곳은 매번 똑같이 하는 지루한 일입니다. 아래에 해당하면 좋은 후보입니다.

🔁 매번 같은 형식

파일 이름 규칙, 보고서 양식처럼 형식이 정해진 일.

📚 양이 많음

파일·행이 수십~수백 개라 손으로 하면 오래 걸리는 일.

📋 규칙이 분명함

"이럴 땐 이렇게"가 명확해 설명할 수 있는 일.

2. 여러 파일을 한 번에 — 정리 · 요약 · 변환

폴더에 파일을 모아 두고, 이렇게 요청해 보세요. (연습용 가짜 파일로)

Claude Code — 예시 요청 
> 이 폴더 파일 이름을 '연도-월-제목' 형식으로 통일해줘
> 이 설문 결과(csv)를 핵심 3가지로 요약해서 표로 만들어줘
> 이 회의록들을 한 페이지 요약본으로 합쳐줘

3. 실습 따라하기 — 신청 명단을 부서별로 집계

실습자료/교육신청명단.csv를 부서별 인원으로 집계해 봅니다. (연습용 더미데이터)

① 이렇게 요청

Claude Code
> 이 폴더의 교육신청명단.csv를 부서별 인원 수로 집계해줘.
  같은 사람이 중복 신청한 건 한 번만 세고,
  결과를 표로 정리해서 부서별집계.md로 저장해줘.

② 이런 결과 (예시)

인사팀 — 3명
교육팀 — 3명
디지털전략팀 — 3명
경영지원팀 — 2명
합계 — 11명

검증 팁: 원본은 12줄인데 홍길동이 2번 신청 → 중복을 빼면 11명이 맞습니다. 이렇게 숫자를 직접 검산하는 게 검증 습관입니다(7단계).

4. 한 번 만들면 계속 쓰는 "작은 도구"

매달 반복하는 계산·정리라면, 아예 작은 도구(웹페이지)로 만들어 두면 다음부터는 클릭만 하면 됩니다.

🧮 교육비 계산기

인원·시간을 넣으면 비용·이수시간이 자동 계산되는 페이지.

📊 설문 대시보드

결과 파일을 넣으면 그래프·요약이 나오는 페이지.

5. 미니 프로젝트 — 내 업무 흐름 설계

내 반복 업무 하나를 골라 입력 → 처리 → 출력의 세 칸으로 정리해 보세요. 이게 곧 클로드 코드에게 줄 요청이 됩니다.

입력

무엇을 넣는가?

내 업무 1건, 흐름으로 적어보기

내 반복 업무 하나를 입력 → 처리 → 출력로 적고, 그걸 그대로 클로드 코드에 요청해 계획을 받아 보세요. 6단계 목표 달성입니다.
