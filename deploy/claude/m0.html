오리엔테이션 & 안전하게 쓰기

본격적으로 시작하기 전에, 오늘 무엇을 하는지와 지켜야 할 안전 규칙을 함께 정리합니다. 딱 20분이면 됩니다.

🎯 이 단계를 마치면
클로드 코드를 안전하게 쓰는 규칙을 내 말로 진술할 수 있습니다.

1. 오늘 우리가 갈 길

하루 동안 아래 흐름으로 진행합니다. 지금 완벽히 몰라도 됩니다 — 순서대로 따라오면 됩니다.

실습 전 딱 하나 확인: 작업 폴더에서 claude가 실행되나요? 안 되면 1단계(설치)로 돌아가 claude --version부터 확인하세요.

2. 가장 중요한 약속 — 더미데이터 원칙

'더미(dummy)데이터'는 진짜가 아닌 연습용 가짜 데이터를 뜻합니다. 오늘 실습에서는 실제 데이터를 쓰지 않습니다.

✅ 써도 되는 것

가상의 이름(홍길동), 임의의 숫자, 공개된 예시, 스스로 만든 연습용 표.

🚫 넣지 말 것

실제 개인정보(주민번호·연락처), 사내 기밀·미공개 자료, 고객 데이터.

왜? 학습 목적이므로 실제 민감정보가 필요 없고, 실수로도 유출되지 않게 하기 위함입니다. "연습은 가짜 데이터로" — 이 한 줄만 기억하세요.

3. AI는 완벽하지 않습니다

클로드 코드는 강력하지만, 가끔 틀리거나 그럴듯한 거짓을 말하기도 합니다. 이것을 알고 쓰는 것이 안전의 핵심입니다.

실수할 수 있음

숫자·사실을 틀릴 수 있으니 결과를 확인하세요.

최신이 아닐 수 있음

모르는 것은 모른다고 하지 않고 지어낼 때가 있습니다.

판단은 사람이

최종 결정과 책임은 사용자에게 있습니다.

4. 안전 사용 3원칙

오늘 하루, 그리고 현업에서도 이 3가지만 지키면 됩니다.

1
연습은 더미데이터로

실제 개인정보·기밀은 넣지 않는다.

2
결과는 반드시 확인

AI가 만든 결과를 그대로 믿지 말고, 사람이 검토한다.

3
막히면 캡처해서 질문

오류 화면을 캡처해 강사(또는 AI)에게 그대로 보여주면 해결이 빠르다.
