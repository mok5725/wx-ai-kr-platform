스킬 라이브러리

확장 스킬로 능력 더하기

클로드 코드는 '스킬(플러그인)'을 설치해 능력을 늘릴 수 있습니다. 각 스킬을 설치법·원리·장단점·사용법·예시로 자세히 다룬 전용 장으로 정리했습니다. 카드를 눌러 각 장으로 이동하세요.

🎯 이 부록에서
필요한 스킬을 골라 설치하고 무엇을 위한 도구인지 설명할 수 있습니다. (아래 추천도는 클로드 코드 사용자 전반 기준의 참고치입니다.)

먼저 — 스킬은 어떻게 설치하나요?

대부분 클로드 코드 안에서 /plugin 명령으로 설치합니다. (터미널이 아니라, 클로드와 대화하는 칸에 /로 시작하는 명령을 칩니다.)

기본 패턴 
/plugin marketplace add [제작자/저장소]   ← 스킬 상점 등록
/plugin install [스킬이름]@[상점]         ← 설치
/plugin                                   ← 설치된 스킬 확인·관리
⚠️ 안전 주의: 스킬은 남이 만든 코드를 내 컴퓨터에서 실행합니다. 신뢰할 수 있는 출처인지 확인하고, 회사 PC라면 사내 정책을 먼저 확인하세요. 아래 정보는 2026-07-07 공개 저장소 기준이며, 설치 전 각 저장소에서 최신 정보를 재확인하는 습관을 권장합니다.
