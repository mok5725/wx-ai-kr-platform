# 실습자료 (연습용 더미데이터)

이 폴더의 파일은 모두 **연습용 가짜 데이터**입니다. 실제 개인정보·업무 데이터가 아닙니다.
교육 실습(5·6단계)에서 클로드 코드에게 "이 파일을 정리·요약·변환해줘"라고 요청할 때 사용하세요.

| 파일 | 용도 | 실습 |
|---|---|---|
| `교육신청명단.csv` | 부서별 인원 집계·중복 제거 연습 | 6단계 직무 응용 |
| `만족도설문.csv` | 결과 요약·설문 대시보드 연습 | 5·6단계 |
| `수료자명단.csv` | 수료증 생성기 입력 데이터 | 5단계 워크플로우 |

> ⚠️ 실습에는 **실제 데이터를 넣지 않습니다.** 연습은 이런 가짜 데이터로 합니다. (2단계 안전 사용 원칙)
