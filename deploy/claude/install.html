클로드 코드 설치하기

명령어는 "복사 → 붙여넣기"가 전부입니다.

검은 화면(터미널)이 낯설어도 괜찮습니다. 이 문서의 명령어는 복사 버튼을 눌러 그대로 붙여넣기만 하면 됩니다. 순서대로 따라오면 claude --version 성공 화면을 보게 됩니다.

✅ 이 문서는 Windows 10/11 기준이며, 명령어는 공식 문서(code.claude.com/docs)로 검증했습니다.

0

준비물 확인

시작 전에 아래 3가지만 있으면 됩니다.

🪟 Windows 10 또는 11

64비트 PC. 관리자 권한이 없어도 대부분 설치됩니다.

🌐 인터넷 연결

설치 파일을 내려받고 로그인할 때 필요합니다.

🔑 Claude 계정

Claude Pro/Max 구독이 필요합니다. (또는 Anthropic Console API 키)

1

PowerShell 열기

"PowerShell"은 명령어를 입력하는 검은 창입니다. 여는 방법은 두 가지 — 편한 걸로 하세요.

방법 A — 시작 메뉴

키보드 Windows 키를 누르고 powershell 이라고 친 뒤 Enter.

방법 B — 탐색기 주소창 (추천)

폴더 창(탐색기) 위쪽 주소창을 클릭해 아래 글자를 치고 Enter. 그 폴더에서 바로 PowerShell이 열립니다.

탐색기 주소창
powershell

▸ 창 맨 앞이 PS C:\ 로 시작하면 PowerShell입니다. (C:\ 만 있으면 CMD — 명령이 다를 수 있어요.)

2

클로드 코드 설치 (공식 권장 방식)

아래 한 줄을 복사해 PowerShell에 붙여넣고 Enter. 자동으로 설치되고, 이후 업데이트도 알아서 됩니다. (이 방식은 Node.js 없이 설치됩니다.)

PowerShell
irm https://claude.ai/install.ps1 | iex
💡 설치가 끝나면 PowerShell 창을 닫았다가 다시 열어 주세요. 새로 설치된 명령을 인식하려면 창을 새로 열어야 합니다(PATH 반영).
다른 설치 방법 (선택)
npm

npm install -g @anthropic-ai/claude-code
Node.js 22 이상 필요. 수동 업데이트.

winget

winget install Anthropic.ClaudeCode
수동 업데이트.

Claude Desktop 앱

GUI로 Claude Code 포함.
단, 터미널 claude 명령은 별도 CLI 설치 필요. Windows는 Git 설치 필요.

3

로그인

아래를 입력하면 브라우저가 자동으로 열립니다. Claude 계정으로 로그인하고 "허용"을 누르면 끝.

PowerShell
claude

▸ 브라우저가 안 열리면 터미널에서 c 를 눌러 로그인 주소를 복사한 뒤 브라우저에 붙여넣으세요.
▸ 계정은 Claude Pro/Max 구독(개인 권장) · Team/Enterprise · Anthropic Console(API) 중 하나면 됩니다.

4

설치 확인 — 이 화면이 나오면 성공!

버전 번호가 보이면 설치 완료입니다. 문제가 있으면 claude doctor가 원인을 알려줍니다.

PowerShell
claude --version    # 버전이 나오면 성공
claude doctor       # 설치·설정 상태 진단

▸ --version은 설치 버전을, doctor는 설치·PATH·업데이트 상태를 점검합니다.

5

내 작업 폴더에서 실행하기

클로드 코드는 "지금 열려 있는 폴더"의 파일을 다룹니다. 그래서 작업할 폴더에서 실행하는 게 핵심입니다. 탐색기 주소창을 쓰면 가장 쉽습니다.

  1. 작업할 폴더를 탐색기에서 엽니다.
  2. 위쪽 주소창을 클릭하고 powershell 입력 → Enter. (그 폴더에서 PowerShell이 열림)
  3. 이어서 claude 를 입력 → Enter. 이제 그 폴더에서 대화를 시작할 수 있습니다.
PowerShell (내 작업 폴더)
claude
done

설치 완료!

이제 클로드 코드를 쓸 준비가 끝났습니다. 위 확인 단계(claude --version)가 통과했다면 다음 단계로 넘어가세요. 잘 안 되는 부분은 화면을 캡처해 두면 해결이 빠릅니다.

함께 설치하면 좋은 도구 (선택 · 심화)

클로드 코드만으로도 실습은 가능합니다. 아래는 더 넓은 작업(파일 버전 관리, 데이터, GitHub 연동)을 하고 싶을 때 추가로 설치하면 좋은 것들입니다. 명령은 PowerShell에 붙여넣으면 됩니다(winget 사용).

🧰 Git for Windows

파일 버전 관리 + Git Bash. 클로드 코드에 필수는 아니지만(없으면 PowerShell을 셸로 사용), GitHub 연동·Bash 도구엔 필요합니다.

PowerShell
winget install --id Git.Git -e

winget이 안 되면 git-scm.com에서 설치 파일을 받아 "다음 → 다음"으로 설치.

📦 Node.js (LTS)

npm으로 각종 스킬을 설치할 때 필요합니다. 안정적인 LTS 버전을 설치하세요.

PowerShell
winget install --id OpenJS.NodeJS.LTS -e

winget이 안 되면 nodejs.org에서 LTS를 받아 설치. 설치 후 node -v 로 확인.

🔗 GitHub CLI

에이전트가 GitHub와 연동해 작업할 때 사용합니다. 설치 후 로그인까지 하면 됩니다.

PowerShell
winget install --id GitHub.cli -e
gh auth login

gh auth login 실행 후 안내가 나오면 방향키로 선택하고 Enter → 브라우저에서 GitHub 로그인.

📝 코드 에디터 (VSCode 등)

코드를 보기 좋게 편집하고, 에디터 안에서 터미널을 열어(Ctrl+`) claude를 바로 실행할 수 있습니다. 원하는 에디터를 홈페이지에서 내려받아 설치하세요. (예: VSCode — 확장: Python·Jupyter)

※ 데이터·AI 실습까지 하려면 Python도 추가하세요 — winget install --id Python.Python.3.12 -e (또는 python.org에서 설치, 이때 "Add Python to PATH" 체크).

교육용 노트북 → 개인 노트북으로 이어하기

가이드 문서 하나만 가져가면 개인 노트북에서도 에이전트가 알아서 세팅해 줍니다.

1
교육용 노트북에서 — 가이드 문서 만들기

아래를 교육용 노트북의 에이전트에 그대로 입력하세요.

Claude Code
현재 프로젝트를 GitHub에 올려줘.
이후에 다른 컴퓨터에서 이 프로젝트를 이어서 작업할 수 있도록,
아래 내용을 포함한 가이드 문서를 만들어줘.
- 이 프로젝트의 GitHub 주소
- 새 컴퓨터에서 clone하는 방법
- 에이전트에게 문서를 주고 작업을 이어달라고 하는 방법
누구나 쉽게 따라할 수 있는 수준으로 작성해줘.
2
개인 노트북에서 — 가이드 문서를 에이전트에게 주기

세팅을 마친 개인 노트북에서 에이전트를 열고, STEP 1에서 만든 가이드 문서를 그대로 붙여넣으세요. 에이전트가 프로젝트를 자동으로 clone하고 이어서 작업할 준비를 해줍니다. ("이전에 작업하던 내용 파악해줘"라고 하면 코드 흐름까지 읽어줍니다.)
